git基础命令
拷贝项目：git clone <仓库地址>
创建分支：git branch <name>
创建并进入分支： git checkout -b <name>
切换分支： git checkout <name>
查看状态： git status
添加所有文件：git add
提交： git commit -m ‘描述信息’
拉取： git pull
推送： git push
查看分支：git branch --list
查看分支（包含远程分支）： git branch -a
合并其他分支到当前文档： git merge --no-ff “描述” <name>

创建分支：git branch dev
进入分支：git checkout dev  
或直接创建并进入：git checkout -b dev
推送分支到远程：git push     、 git push --set-upstream origin dev
